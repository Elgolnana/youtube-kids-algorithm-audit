# YouTube Kids 알고리즘 감사 실험 — 실행 가이드

## 실험 개요
부모 페르소나(방관형/보통형/적극형)별로 YouTube Kids 알고리즘 오염 및 AI 개입 효과를 측정하는 멀티에이전트 실험입니다.

---

## 준비물

### 필수 파일
- `시드영상_목록.xlsx` — 시드 영상 목록 (유형/제목/URL 컬럼, 팀원에게 별도 공유)
- 나눔스퀘어 폰트 — `~/Library/Fonts/NanumSquare_acR.ttf`, `NanumSquare_acB.ttf` (PDF 한글 출력)

### Python 패키지 설치
```bash
pip install playwright openpyxl reportlab groq pandas matplotlib pillow
playwright install chrome
```

### 필요 계정/프로필
- Google 계정 1개 (YouTube Kids 연동)
- YouTube Kids 자녀 프로필 **반드시 3개**, 이름은 아래 후보 중 하나여야 합니다:

| 창 번호 | 프로필 이름 (YouTube Kids에서 정확히 이 이름으로 생성) |
|---------|------------------------------------------------------|
| 1번 창 | `방관형` |
| 2번 창 | `보통형` |
| 3번 창 | `적극형` |

> ⚠️ 프로필 이름이 위 목록과 다르면 자동 재선택이 작동하지 않습니다.

### API 키
- `GROQ_API_KEY` — Judge Agent(LLM 분석)에 필요. 없으면 실행 불가.

## 파일 구성

| 파일 | 용도 |
|------|------|
| `전체실험_최종.py` | **실제 실험용** (영상당 5분, 유형별 7개 × 4유형 = 28개, 약 2시간 20분) |
| `전체실험_시연용.py` | **시연용** (영상당 10초, 유형별 무작위 7개) |
| `시드영상_목록.xlsx` | 시드 영상 목록 |

---

## 실험 파이프라인 (자동 순차 실행)

```
P1 수집 → 오염(시드 영상 시청) → P2 수집 → AI 개입(차단) → P3 수집 → 분석
```

| 단계 | 내용 |
|------|------|
| P1 수집 | 오염 전 홈피드·비디오페이지 각 10개 수집 + 스크린샷 |
| 오염 | 유형별 7개 × 4유형 = 28개 × 3 프로필 동시 시청 (영상당 5분, 약 2시간 20분) |
| P2 수집 | 오염 후 피드 수집 + 스크린샷 |
| AI 개입 | Judge(Groq)가 P2 분석 → 기초+추가 키워드로 차단 |
| P3 수집 | 개입 후 피드 수집 + 스크린샷 |
| 분석 | P1→P2→P3 부적합 영상 비율 비교, 그래프·PDF 출력 |

---

## 실행 방법

```bash
GROQ_API_KEY="gsk_..." python3 -u "전체실험_최종.py"
```

## 실행 중 사용자 입력

### 1. 로그인 완료 후
브라우저 3창이 열리면 **모든 창에서 동일한 Google 계정으로 로그인** 완료 후:
```
로그인완료
```

### 2. 프로필 선택 완료 후
각 창에서 아래 순서대로 프로필 선택:
- 1번 창 → **방관형**
- 2번 창 → **보통형**
- 3번 창 → **적극형**

선택 완료 후:
```
준비됐어
```

> **주의**: 처음 프로필 선택은 사용자가 직접 합니다. 실험 도중 프로필 화면이 갑자기 뜨면 가디언이 자동으로 재선택합니다.

## 중간에 끊겼을 때 재시작

`실험_진행상황.json`에서 해당 단계 이후를 제거하면 자동으로 그 단계부터 재시작합니다:

```json
{ "completed": ["p1", "contaminate", "p2"] }
```
→ 4단계(AI 개입)부터 재시작

## 페르소나별 차단 방식

| 페르소나 | 차단 단위 | 기초 키워드 | Judge 추가 |
|----------|----------|------------|-----------|
| 방관형(키즈) | 없음 | 없음 | 없음 |
| 보통형(키즈2) | 영상 단위 | 런닝맨·아는형님·이혼숙려캠프·체인소맨·귀멸의칼날·19금·이혼·피어싱·시청주의·경고 | P2 피드에서 고유명사 + 동일 유형 추가 단어 |
| 적극형(키즈3) | 채널 단위 | 보통형 기준 전체 + 주술회전·진격의거인·사춘기·화장·다이어트·성형·공포·귀신·술·담배·연애·남친·여친·엄마 몰래 | P2 피드에서 고유명사 + 더 넓은 유형 추가 단어 |

**최종 차단 키워드 = 기초 키워드 + Judge가 P2 피드 분석 후 추가 도출**

## 알려진 주의사항

- 새 계정일수록 오염 효과가 더 뚜렷하게 나타남
- 시드 영상이 YouTube Kids에 없으면 자동 건너뜀
- 차단 직후 P3 수집 시 홈피드 로드 대기 15초 포함
- 3글자 미만 키워드는 Executor에서 자동 제외 (오차단 방지)
- 디즈니·핑크퐁·뽀로로 등 공인 아동 콘텐츠는 Judge가 키워드에서 제외하도록 지시됨
